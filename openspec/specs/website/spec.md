# Website — Capability Specification

## Purpose

The Laraskills website is the public-facing showcase for the Laraskills product. It presents Laraskills as a serious open-source developer tool, explains what it does and who it helps, and makes installation and onboarding easy.

## Current State

The website is initialized as a clean Next.js 16 project with:
- App Router
- TypeScript
- Tailwind CSS v4
- ESLint
- A minimal homepage placeholder displaying the Laraskills name and tagline
- Vercel-ready configuration

No landing page sections, documentation pages, or feature content have been built yet.
